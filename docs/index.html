Medli2

Medli ‘translated’ to .NET Core

Medli OS is an operating system developed in C# using the Cosmos toolkit.

I am currently working on developing a new kernel for Medli, with a new shell and increased support for command line arguments, more IO functionality and error handling.

For the time being, this new kernel is going to be closed source but I will be working on both the existing Medli kernel/shell, and this new one, so development between the two will be split.
