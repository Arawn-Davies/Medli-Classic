Help

Medli's help command allows you to view what commands are accepted in Medli's shell. As more commands are implemented, this list will be updated. Currently, Medli is aiming to have a BASH-like shell which will allow you to store variables and use them in scripts later.

To see more documentation on what is going on inside Medli, click this link to view the documentation generated using Doxygen.

A doxyfile is included in the /docs directory so you can generate documentation in your desired format using Doxygen.
